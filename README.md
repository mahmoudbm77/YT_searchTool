# Project Name
Search Tool

## Description
This is a simple web-based project that allows users to enter a search query and be redirected to YouTube with the corresponding search results.  
The main goal of this project is to demonstrate responsive web design principles and basic user interaction through forms.

## Features
- Responsive design for all devices
- Simple and clean interface
- Easy user interaction

## How to Use
1. Open the project in a browser
2. Type a search query in the input field
3. Click "Search"
4. You will be redirected to YouTube with the search results
